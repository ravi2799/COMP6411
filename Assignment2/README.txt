************* Folder structure *********************

Java and Python folder contains respective files for languages.

---java
|       BinarySearchTree.java              ----------------> java file contains code for binary search tree
|       BinarySearchTree_txt.txt           ----------------> binary search tree code in txt file
|       Binary_Search_tree_output_java.txt ----------------> output of binary search tree 
|       HashTable.java                     ----------------> java file contains code for the hashatable
|       HashTable_output_java.txt          ----------------> output of hashtable
|       HashTable_txt.txt                  ----------------> hashtable code in text file
|       rand.txt                           -----------------> Input file for the code
|       
---Python
        binary_search_python.py            -----------------> Python file contains code for binary search tree
        Binary_Search_tree_python_output.txt ---------------> binary search tree output 
        Binary_Search_tree_python_txt.txt    ---------------> binary search tree code in txt file
        hash_table_output_python.txt        ----------------> output of hash table      
        hash_table_python.py                ----------------> python file contains code of hash table
        hash_table_python_txt.txt           ----------------> hash table code in text file
        rand.txt                            ----------------> Input file for the code

        
------------------- For run the program --------------------------

----------- **PYTHON** ----------
Step1:  python binary_search_python.py (This will generate Binary_Search_tree_python_output.txt which contains the statistics)
Step2: python hash_table_python.py (This will generate HashTable_output_java.txt  which contains the statistics )


----------- **JAVA** ----------

Step1: javac *.java
Step2: java BinarySearchTree  (This will generate Binary_Search_tree_output_java.txt which contains the statistics)
Step3: java HashTable (This will generate HashTable_output_java.txt which contains the statistics)
